# RFM-анализ и когортная аналитика e-commerce проекта

Kомплексный анализ клиентской базы e-commerce бизнеса. Были выполнены два ключевых блока аналитики:

1. RFM-анализ клиентской базы
2. Когортный анализ, расчет выручки и LTV

Определена структура выручки, качество клиентской базы и модель роста бизнеса.

---

## Используемые данные

Транзакционный e-commerce датасет со следующими полями:

- customer_unique_id — уникальный клиент
- order_purchase_timestamp — дата покупки
- order_id — заказ
- payment_value — сумма заказа
- order_status — статус заказа

Анализ выполнялся только по доставленным заказам.

---

## Блок 1. RFM-анализ

Для каждого клиента рассчитаны:

- Recency — давность последней покупки
- Frequency — количество заказов
- Monetary — суммарная выручка

Далее:
- выполнена сегментация клиентов
- выделены высокодоходные сегменты
- проведен Pareto-анализ (20/80)
- построена кривая Лоренца
- рассчитан коэффициент Джини

### Основные выводы RFM

- 20% клиентов формируют 53% выручки
- Коэффициент Джини 0.47 — умеренная концентрация
- Ярко выраженного VIP-ядра нет
- Бизнес не критически зависит от узкого сегмента клиентов

---

## Блок 2. Когортный анализ

### Построено:

- Матрица удержания
- Матрица выручки по когортам
- Накопительная выручка
- LTV по когортам

### Ключевые выводы

- Основная выручка формируется в первый месяц (M0)
- Удержание слабое, повторная монетизация минимальна
- Средняя выручка на клиента в M0 стабильна между когортами
- Рост общей выручки обеспечивается масштабом привлечения, а не удержанием

---

## Общий стратегический вывод

Бизнес-модель acquisition-driven:

- доход формируется в первый месяц,
- повторные покупки играют незначительную роль,
- масштабирование происходит за счет увеличения размера когорт.
